Задача:
Нужно создать Telegram Mini App / веб-портфолио для дизайнера, которое визуально и UX-похоже на главную страницу YouTube, но это не копия YouTube, а авторское портфолио в похожей логике интерфейса.
Проект должен быть:
статическим
без backend
под GitHub Pages + custom domain
с возможностью потом использовать как Telegram Mini App
удобным для ручного наполнения через JSON
оптимизированным под русскоязычную аудиторию, но с поддержкой английского языка
хорошо работающим на мобилке, так как основное открытие будет через Telegram
1. ТЕХНОЛОГИИ
Использовать:
React
Vite
JavaScript (можно TypeScript только если структура будет очень чистой и понятной)
CSS modules или SCSS, либо Tailwind, если это действительно ускорит и не превратит код в мусор
React Router использовать только если это реально нужно; предпочтительно сделать всё как single-page app без сложного роутинга
GitHub Pages как хостинг
проект должен быть полностью статическим
никакого backend, базы данных, CMS и лишней серверной логики
2. ОСНОВНАЯ ИДЕЯ ПРОЕКТА
Это портфолио в стиле главной YouTube.
Пользователь открывает сайт и видит:

под поиском блок фильтрации по тегам
справа сверху:

круглая аватарка владельца портфолио
ниже — сетка карточек работ
Сайт должен визуально напоминать YouTube home feed, но без слепого копирования бренда.
Это должна быть стилизованная дизайнерская интерпретация.

теги

“Видео”

или фильтр по типу контента
